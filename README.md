1. Кратко описание на добавената функционалност.
    - Проектът съдържа всички разработки направени по време на упражненията + домашната работа за добавяне на прогресивен лихвен план.
    - Предлага се възможност за избор между фиксирана лифва или лихвен план, като фиксираната лихва се опраделя по време на направата на депозита,
    а лихвеният план по време на изтеглянето.
    - Лихвеният план поддържа 3 прага, които се определят от 2 зададени стойности за време.
    - Всички добавени от нас разработки са обяснени с коментари на английски език.
